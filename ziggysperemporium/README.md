# Ziggy's Pet Pamporium

A modern, pet-friendly marketing site for Ziggy's Pet Pamporium in Melville, Johannesburg. Built with Next.js (App Router) and TypeScript, with Markdown-driven content and motion-enhanced UI.

## Tech

- Next.js with App Router
- TypeScript
- React Markdown for content rendering
- Framer Motion for micro-animations

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run dev server:

   ```bash
   npm run dev
   ```

3. Lint & type-check:

   ```bash
   npm run lint
   npm run typecheck
   ```

## Content

Content and SEO metadata live in `content/siteContent.ts` and are consumed via Markdown renderers across pages.
